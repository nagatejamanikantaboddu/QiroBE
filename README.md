# qiroBE

Backend starter template for a chat application built with Express.js.

## Tech Stack

- Node.js (ES modules)
- Express.js
- MongoDB + Mongoose
- Redis
- JWT
- Winston (logging)
- Morgan (request logging)

## Folder Structure

```
src/
  ├── config/          # Configuration and utilities
  ├── middlewares/     # Custom middleware
```

## Status

Active development. README updated continuously alongside codebase changes.

## Getting Started

```bash
yarn install
yarn start      # Production start
yarn dev        # Development with watch mode
```
